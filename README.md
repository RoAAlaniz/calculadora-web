# calculadora-web
Aplicación web que simula una calculadora funcional.

## 📌 Descripción

Este proyecto consiste en una calculadora web funcional desarrollada como práctica de lógica de programación y manipulación del DOM. 

Permite realizar operaciones matemáticas básicas a través de una interfaz interactiva y responsive. El objetivo fue aplicar conocimientos fundamentales de desarrollo frontend utilizando tecnologías base de la web.

---

## 🛠 Tecnologías utilizadas

- HTML
- CSS
- JavaScript

---

## ⚙️ Funcionalidades

- Ingreso de números mediante botones interactivos.
- Operaciones básicas: suma, resta, multiplicación y división.
- Visualización dinámica del cálculo en pantalla.
- Botón para limpiar la operación.
- Manejo básico de errores ante operaciones inválidas.

---

## 🚀 Objetivo del proyecto

Practicar la lógica de programación en JavaScript y reforzar conceptos de estructura (HTML) y diseño (CSS), integrando todo en una aplicación web funcional.
